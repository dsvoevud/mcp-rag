This is a sample markdown document used for smoke-testing the RAG indexer.

## What is Machine Learning?

Machine learning is a subset of artificial intelligence that enables systems to
learn and improve from experience without being explicitly programmed.

## Types of Machine Learning

- **Supervised learning** — trained on labelled data
- **Unsupervised learning** — finds patterns in unlabelled data
- **Reinforcement learning** — learns via rewards and penalties
